  • TAP-SWOT in a box

    TAP-SWOT is a co-operative serious card game about SWOT analysis approach to a sale pitch. This game is a part of university works made by a group of students. I responsible in Game Design Document, UI flow design, UX design and research and created flow of the game with my colleagues. Additionally, I also deal with UI Art and Design including typonograhy and iconography and developed in Unity. My colleagues focus on gameplay programming and network programming respectively.

    The flow of the game done with Miro Flowchart and mock-up as UI/UX prototype. Low-fidelity Wireframe created in Adobe XD and Hi-fidelity Wireframe created in Figma. My iconography created from Adobe Illustrator reference from uxwing.com. Some assets such as background created from Adobe Photoshop and Illustrator.

    This project also included game user research using surveys, observation, questioning and herustic evaluation. As well as quality evaluation of serious game according to learning achievements, enviroment engagement and user's motivation.

  • Labyrinth of Chroma

    Labyrinth of Chroma is my solo and personal project. This game collide between the match 3 shooting game and hell bullet shooting game together. I do everything from art, game mechanics design and balanced, animation and code. Audio and sound is picked from the free stock audio. The challenge of this design is a balancing challenge between hell bullet shooting games and match 3 shooting game and convey the UX design that confortable to player's eye focus.

    This game prototype developed in Unity. Art and assets is done in Photoshop. Numerical design and balance mechanics using Microsoft Excel.

  • Temporal Tenacity

    Temporal Tenacity is a point-and-click shooting game where people can travel accross the time. This game is a part of university works made by a group of eight students. I responsible in UX design and layout. Additionally, I also resonspible in UI Art and Design included iconography and create wireframe.

    The layout inspired from the point-and-click game similiar to this game such as Diablo and DOTA. My iconography created from Adobe Illustrator reference from uxwing.com and Wireframe created in Figma.
